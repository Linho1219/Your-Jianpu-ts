# Your-Jianpu-ts

Haskell 项目 [Your-Jianpu](https://github.com/speedyOrc-C/Your-Jianpu/) 在 TypeScript 的移植版本。本项目尝试使用 TypeScript 将自定义的音乐数据结构转换为 **印刷质量的 SVG 简谱谱面**。

与 [OpenFanqieCore](https://github.com/Linho1219/OpenFanqieCore) 同为使用 TS 渲染简谱谱面的尝试。

- 本仓库所说简谱，是指主要中国大陆地区通行的、以数字表示音符的乐谱形式；
- 由于简谱主要在中国大陆地区通行，使用者通常使用中文，本仓库**采用中文为工作语言**。

---

**For non-Chinese speakers:**

This repository is a TypeScript port of [Your-Jianpu](https://github.com/speedyOrc-C/Your-Jianpu/), and aims to render **[numbered musical notation](https://en.wikipedia.org/wiki/Numbered_musical_notation)** into SVG.  

Since numbered notation is mainly used in **mainland China**, and most users are native Chinese speakers, the working language of this repository is **Chinese**.  

If you are not involved with numbered notation, this repo is probably irrelevant to you.
